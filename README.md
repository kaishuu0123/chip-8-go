# chip-8-go

chip-8-go is [CHIP-8] Emulator written in GO language.

written in Go and uses SDL.

## Usage

```
go run cmd/chip-8-go/main.go roms/BRIX
```

## Key Mapping

|      |      |      |      |
|------|------|------|------|
| 1    | 2    | 3    | 4(C) |
| Q(4) | W(5) | E(6) | R(D) |
| A(7) | S(8) | D(9) | F(E) |
| Z(A) | X(0) | C(B) | V(F) |
